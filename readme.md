# 基于线性回归的ARMIA规律分析
- 使用了随机生成的500个线性分布的数据，合并噪声
- 生成了包含输入数据和规律分析的示意图

## 运行
1. 创建Anaconda虚拟环境：
```
conda create -n ARIMA python=3.8
```

2. 进入环境：
```
conda activate ARIMA
```

3. 进入工程文件夹
```
cd .../.../ARIMA
```

4. 安装依赖
```
pip install -r requirements.txt
```

5. 测试
```
python main.py
```
